## Parametric maps and Spectroscopy Tool (PST)

This MATLAB tool allows LCModel processing and obtaining the T1w-segmentation and qMRI maps in Philips and Siemens 2D MRSI and single voxel MRS.

### Supported OS
- MS Windows
- Unix (not the LCModel part)
- Mac OS (not the LCModel part)

Full compatibility is being developed.

### How to run the GUI
Run PST.m in MATLAB

### How to use
Please refer to INSTRUCTIONS.m file.

### Supported file formats
- Images: .NII
- MRS/MRSI: SDAT/SPAR, RDA
